## Description

Python script to help automate the task of splitting a video file into smaller parts using ffmpeg.

## Usage

- Modify the csv file and set the actual file names and timestamps
- Run the script:  
```
python3 csv_to_ffmpeg_cmd.py  
```

## Dependencies
- ffmpeg
